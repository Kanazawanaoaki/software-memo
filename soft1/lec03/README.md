破壊的操作と非破壊的操作  
```python
In [1]: a = [2,7,4,2,1,6]

In [2]: a[::-1]
Out[2]: [6, 1, 2, 4, 7, 2]

In [3]: a
Out[3]: [2, 7, 4, 2, 1, 6]

In [4]: list(reversed(a))
Out[4]: [6, 1, 2, 4, 7, 2]

In [5]: a
Out[5]: [2, 7, 4, 2, 1, 6]

In [6]: a.reverse()

In [7]: a
Out[7]: [6, 1, 2, 4, 7, 2]
```
`a.sort()`破壊的操作,`sorted(a, reverse=True)`降順でソート
